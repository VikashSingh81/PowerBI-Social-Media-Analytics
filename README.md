## 📌 Project Overview

This project presents an end-to-end **Power BI Dashboard** developed to analyze **700+ student records** and identify how excessive social media usage influences **sleep quality, mental health, and academic performance**.

The dashboard transforms raw data into interactive visual insights using **Power Query**, **DAX**, and **Power BI**, allowing users to explore behavioral patterns, addiction levels, platform usage, and key influencing factors through dynamic reports.

---

## 🎯 Problem Statement

Excessive social media usage has become one of the biggest challenges among students today. Increased screen time can negatively affect sleep, mental health, and academic performance.

The objective of this project is to analyze social media addiction data and build an interactive dashboard that helps identify important trends, patterns, and factors contributing to digital addiction.

---

## 🎯 Objectives

- Analyze social media addiction trends.
- Study the relationship between screen time and sleep.
- Understand the impact on academic performance.
- Identify factors affecting mental health.
- Build an interactive Power BI dashboard for decision-making.

# 🧹 Data Preprocessing

Before creating the dashboard, the dataset was cleaned and transformed using **Power Query**.

The preprocessing included:

- Removing unnecessary columns
- Handling missing values
- Formatting data types
- Creating calculated columns
- Creating Age Groups
- Creating Addiction Categories
- Preparing the dataset for dashboard visualization

# 📈 Key Insights

After analyzing the dataset, the following insights were observed:

- 📱 Students with higher daily screen time tend to have higher social media addiction levels.
- 😴 Reduced sleep duration is strongly associated with excessive social media usage.
- 🧠 Increased addiction levels are linked with higher anxiety and depression scores.
- 🎓 Academic performance shows a declining trend as addiction levels increase.
- 🤖 Power BI's **Key Influencers** visual identified screen time and sleep hours as the major contributors to addiction.
